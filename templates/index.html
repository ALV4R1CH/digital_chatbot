Chatbot de Asesoramiento Digital

Un chatbot impulsado por IA (Hugging Face) para ayudar a negocios a identificar soluciones digitales. Desarrollado con Flask, Socket.IO, SQLite, y Hugging Face Inference API.

  • Tecnologías: Python, Flask, Socket.IO, SQLite, Hugging Face API
  • Funcionalidades: Chat en tiempo real, respuestas inteligentes, almacenamiento de leads
